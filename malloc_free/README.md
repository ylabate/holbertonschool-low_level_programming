# malloc_free

## 0-create_array.c

creates an array of chars and initializes it with a specific char

## 1-strdup.c

duplicates a string using malloc

## 2-str_concat.c

concatenates two strings into a newly allocated space in memory

## 3-alloc_grid.c

allocates a 2D grid of integers initialized to 0

## 4-free_grid.c

frees a 2 dimensional grid previously created by alloc_grid
